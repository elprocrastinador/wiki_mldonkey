Block list

From MLDonkey
(Redirected from Block test)
Jump to: navigation, search

Syntax

~> block_list
~> block_test [<ip>]

Description

block_list displays the list of blocked IP address ranges that were hit by MLDonkey with the description and the number of hits since the load of the block list.

block_test checks if an IP address is blocked by MLDonkey and show the description of the block given in the block list.

The block list can be loaded from a local file and/or a web url in the PeerGuardian P2P or eMule ipfilter.dat formats and may be compressed in the gzip, bzip2 or ZIP formats.

See also

bp, bs, ip_blocking, rem, stats, web_infos
