Max source age

From MLDonkey
Jump to: navigation, search

Unit

  • Before 2.02-8: days
  • 2.02-8: half days (12 hours)

Description

Removes sources that could not be reached for longer than the given delay. With all these DSL-peers around that have dynamic IP, it's a good idea to set this pretty low, so abandoned IPs don't accumulate.
