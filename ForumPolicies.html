MLdonkeyWorld's forums is a technical forum around the use of the MLdonkey software.

They are open for everyone to read, but (free) registration is required for posting. The only public-writable forum is Problems and Suggestions for this site, so you can ask for help there if you have trouble registering or logging in.

This is not a warez site. Please refrain from posting names (or worse, p2p links) of copyrighted materials. Offending messages will either be edited, or simply deleted by moderators, depending on their mood.

When reporting problems, be sure to give all informations that could be relevant, like MLdonkey version you used, its origin (precompiled or self-compiled), the platform, and possibly hardware details too.

Be sure to write in the right forum for your post:

Mulus

Mulus is a fork of MLdonkey, that has its roots in MLdonkey 2.5.16.

Starting with MLdonkey 2.5.17, legacy eDonkey code was heavily modified, to use more code common to all networks (the swarmer, for example).

While it was necessary for multi-network support improvement, several features were dropped, and new support had lower performance than the old, proven one, so many people only interested in eDonkey support sticked with version 2.5.16.

Knocker then created the Mulus branch from this version, in order to further improving it, and also backport some features from the official branch.

At the time of this writting, official branch's eDonkey support has since then mostly recovered (some features are still missing), and Mulus development seems to have stalled.
