Keyboard Shortcuts for the Gtk GUI (mldonkey_gui)

here are the shortcuts as defined in my .mldonkey_gui.ini file. the shortcut support seems to be somewhat buggy at the moment (2.02.9).

Global Shortcuts

these open the respective page:

these two are preferred, because they always work:

  • alt - <right> : next page
  • alt - <left> : previous page
  • alt - s : server
  • alt - d : downloads
  • alt - f : friends

alt - c : console ~ alt - h : help ~ alt - q : queries ~

  • alt - o : options

these are supposed to work, but the ones marked with an ~* do work differently (at least with the GUI that came with 2.02.9):

  • alt - c : help
  • alt - q : results
  • alt - h : did not work.
  • alt - q : results

these perform an action:

  • ctl - r : reconnect to core
  • ctl - e : quit gui
  • ctl - q : quit gui

Key bindings for servers tab

  • ctl - a : select all
  • ctl - m : connect more
  • ctl - c : connect

Key bindings for downloads tab

  • ctl - a : select all
  • ctl - s : save-file menu
  • ctl - shift - s : save all files
  • ctl - c : cancel download

Key bindings for friends tab

  • ctl - a : select all
  • ctl - x : remove friend
  • ctl - d : download selection
