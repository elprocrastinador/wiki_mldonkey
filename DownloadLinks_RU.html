Использование CVS и SVN

Как получить исходный код MLDonkey с CVS? 
cvs -z3 -d:pserver:anonymous@cvs.savannah.nongnu.org:/sources/mldonkey co -P mldonkey
Как получить исходный код OCaml с SVN? 
svn co http://caml.inria.fr/svn/ocaml/release/3.12.0 ocaml-3.12.0


Для разработчиков

  • Developing Applications With Objective CAML, эта книга рекомендована пользователем pango
  • Initiation à la programmation fonctionnelle (French)
Вступление к функциональному программированию с примерами на OCAML (на французском).
  • Introduction to the Objective CAML Programming Language
Хорошее начало изучения OCAML для людей, которые уже знают C и подобные языки.
  • Objective CAML Wiki tutorial
  • Structure and Interpretation of Computer Programs
Превосходная онлайн-книга для понимания парадигм функционального программирования. Используется Scheme, но все концепции применимы ко всем функциональнм языкам, включая OCAML, естественно ;)
  • tools for ed2k
ed2k-tools - коллекция небольших вспомогательных приложений для программ, работающих с eDonkey2000 и Overnet.
