Small files slot limit

From MLDonkey
Jump to: navigation, search

small_files_slot_limit is the maximum file size to benefit from the reserved slot for small files. All files below the configured file size, expressed in Bytes, are uploaded through one reserved slot. Setting it to 0 disable this feature.

Default value

small_files_slot_limit is 10240 (10 kBytes).

See also

dynamic_slots, friends_upload_slot, max_release_slots, max_upload_slots, nolimit_ips
