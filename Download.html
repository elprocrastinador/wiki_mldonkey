Third-party packages

NB may not have the latest version
NB usually there are two separate packages for mldonkey - core and gui

Debian, Ubuntu aptitude install mldonkey-server
Fedora yum install mldonkey-server
openSUSE  ?
Gentoo emerge net-p2p/mldonkey
Mac OS X MacPorts port install mldonkey
FreeBSD portinstall -s net-p2p/mldonkey-core
OpenBSD pkg_add -iv mldonkey
NetBSD cd /usr/pkgsrc/net/mldonkey && make install
Sabayon equo install mldonkey --ask
Arch pacman -S mldonkey

Development version

(use this for patches)

Development now happens on github : 

 git clone git@github.com:ygrek/mldonkey.git

There is another git mirror at repo.or.cz : 

 git clone git://repo.or.cz/mldonkey.git

Older CVS repository hosted by Savannah : 

 cvs -z3 -d:pserver:anonymous@cvs.savannah.nongnu.org:/sources/mldonkey co -P mldonkey
