Costats

From MLDonkey
Jump to: navigation, search

Syntax

~> costats [<all|regex>]
~> countries
~> reset_costats

Description

costats displays country based statistics with the amount of data transfered and the number of clients seen. Without parameter it displays all countries where data was transfered from/to. The optional argument can be all to display all known countries or a regular expression to select country name, code and continent. For example the regular expression *aus* will show Austria and Australia, *eu* all europeans countries and Reunion, oceania de* all oceanians countries, Germany (code DE) and Denmark.

countries displays the country database with country name, code and continent.

reset_costats resets the country based transfer statistics.

Examples

~> costats fr
Country Code  Con    sUL  sDL  sSeen  tUL   tDL  tSeen 
France  FR   Europe 24.5M 1.8M  2042 2.99G 2.37G 86394

~> costats *nor* de*
   Country    Code      Con       sUL   sDL  sSeen  tUL    tDL   tSeen 
Canada        CA   North America  2.2M     0     2  1.54G 246.5M  9085
Germany       DE   Europe        29.0k     0    72  1.05G 653.4M 93331
Denmark       DK   Europe            0     0     0 128.3M  27.7M  5657
Mexico        MX   North America 19.1M  9.7M   235  1.25G 616.1M  8418
Norway        NO   Europe            0     0     0  27.0M 324.6M  1789
United States US   North America 12.4M 26.0k    37  6.27G  2.49G 32155
