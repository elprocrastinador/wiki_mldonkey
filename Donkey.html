Intro

All options that are found in donkey.ini file are mentioned here. Remember, - do not edit configuration files while mldonkey is running because your changes will be lost. Either you use one of the GUI's or you use the telnet interface. All options from donkey.ini need to be specified with the prefix "ED2K".

For example - to change the port option using the telnet interface or the commandline in the webinterface: 

> set ED2K_port 4661

If you know something about these options feel free to edit them.
